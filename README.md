# Partner Integration Platform

## Overview

This project is an enterprise-grade Partner Integration Platform.

It will provide:

- Partner Onboarding
- API Management
- 3scale Integration
- OpenShift Deployment
- GitHub Actions CI/CD
- Secure APIs
- Document Management

## Technology Stack

- Angular
- Spring Boot
- PostgreSQL
- Docker
- OpenShift
- Red Hat 3scale
- GitHub Actions
